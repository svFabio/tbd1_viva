# Problema — VIVA Bolivia

VIVA es una empresa boliviana de telefonía móvil. Necesita un sistema de base de datos para gestionar clientes, líneas, recargas, consumos, paquetes y facturación.

---

## ¿Qué hace VIVA que lo hace especial?



### 1. Número Amigo
El cliente puede registrar un número VIVA para llamadas gratuitas o con descuento. Tiene un costo de activación y una fecha de vencimiento.

### 2. Fidelización VIVA BONUS
Por recargar, navegar o referir amigos, el cliente acumula puntos. Con esos puntos puede canjear premios sin pagar dinero.
